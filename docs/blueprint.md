# **App Name**: LopezWelder PWA

## Core Features:

- Registro de Horas: Registra las horas de inicio y fin de cada día laboral.
- Cálculo Automático: Calcula automáticamente las horas diarias trabajadas según las horas de entrada y salida.
- Ingreso de Tarifa por Hora: Ingresa una tarifa por hora para el cálculo de las ganancias.
- Edición de Registros: Edita, elimina y crea registros de tiempo para fechas pasadas.
- Panel de Informes: Muestra resúmenes diarios, semanales, quincenales y mensuales de horas y ganancias.
- Compartir por WhatsApp: Genera y envía informes a través de WhatsApp al +5548991590325.
- Persistencia de Datos: Integración de almacenamiento local para guardar los registros de trabajo. No se necesita servidor.
- Exportación/Importación de Datos: Permite exportar e importar datos en formato JSON.
- Exportación a PDF: Permite exportar un PDF con el resumen elegido (diario, sem, quin, mens).

## Style Guidelines:

- Esquema de colores oscuros para una visualización cómoda con un fondo #212121.
- Color primario: Naranja rojizo (#FF5722) para un aspecto ardiente.
- Color de acento: Amarillo anaranjado (#FFB300) para elementos interactivos.
- Fuente: 'Roboto' (sans-serif) para encabezados y texto del cuerpo para mantener una interfaz limpia y legible.
- Utiliza iconos simples de estilo lineal para mayor claridad y un aspecto moderno.
- Enfoque de diseño móvil primero, asegurando una experiencia de usuario receptiva e intuitiva en pantallas más pequeñas.
- Transiciones y animaciones sutiles para proporcionar una experiencia de usuario fluida, como el fuego.
- Liquid glass ui
